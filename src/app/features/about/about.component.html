Projeto Valorant

Projeto desenvolvido com o framework Angular 14, consumindo dados de uma REST API pública com informações detalhadas sobre o jogo Valorant.

A aplicação utiliza NgRx para gerenciamento de estado global, garantindo uma estrutura reativa, escalável e performática.

Também foram aplicadas práticas modernas de desenvolvimento front-end, como componentização, responsividade e organização com SCSS modular.
